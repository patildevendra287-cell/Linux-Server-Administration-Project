# Linux Server Administration Project

## Objective
Practice Linux system administration tasks commonly used in IT Support and Linux Administration.

## Tasks Performed

### User Management
- Created users and groups
- Modified user accounts
- Managed passwords

### File Permissions
- Used chmod
- Used chown
- Changed ownership and permissions

### Package Management
- Installed packages
- Updated packages
- Removed packages

### System Monitoring
- Checked CPU usage
- Checked Memory usage
- Checked Disk usage

### Networking
- Checked IP Address
- Tested network connectivity using ping
- Verified hostname

This project improved my Linux administration skills and hands-on knowledge for entry-level IT Support and Linux Support roles.
